# Active Directory Homelab Project
Windows Server 2022 • Active Directory • DNS • DHCP • Group Policy • File Shares • VirtualBox
## Overview

Built a Windows Server 2022 Active Directory environment in VirtualBox to simulate real-world IT Support and Helpdesk tasks.

## Project Objectives

This homelab was built to gain hands-on experience with common IT Support and Helpdesk tasks including:

- Active Directory user administration
- Password resets and account lockouts
- Security group management
- Shared folder permissions
- Group Policy administration
- Domain-joined Windows client management
- DNS and DHCP configuration

## Environment

- Windows Server 2022
- Windows 11 Client
- Active Directory Domain Services (AD DS)
- DNS
- DHCP
- Group Policy Management
- SMB File Shares
- VirtualBox

## Project Tasks Completed

### User and Group Management
- Created Organizational Units (HR, IT, Finance, Sales)
- Created domain users
- Created security groups
- Managed group memberships

### Password Management
- Password reset scenario
- User login verification

### Account Lockout Management
- Configured account lockout policy
- Locked user account after failed logins
- Unlocked account in Active Directory

### Shared Folder Permissions
- Created Finance shared folder
- Configured share permissions
- Configured NTFS permissions
- Verified authorized access
- Verified unauthorized access denial

### Group Policy
- Created and linked a GPO
- Restricted Control Panel access
- Tested policy on domain-joined client

### Domain Administration
- Installed AD DS
- Configured DNS
- Configured DHCP
- Joined Windows 11 to domain
